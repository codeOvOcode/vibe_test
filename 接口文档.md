## 一、选课管理 API

### 1.1 CSV 批量导入选课数据

导入 CSV 格式的选课数据，后端执行去重 → 排序 → 分类处理后返回。

| 属性 | 值 |
|------|-----|
| **方法** | `POST` |
| **路径** | `/api/enrollment/import` |
| **Content-Type** | `application/x-www-form-urlencoded` |

**请求参数** (form-urlencoded):

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `csvData` | String | 是 | CSV格式选课数据，每行一条，格式：`学生ID,课程ID,课程名称,课程类型` |

**CSV 格式示例**:
```
S000001,C000001,Java程序设计,专业课
S000002,C000003,计算机网络,公共课
S000001,C000001,Java程序设计,专业课
S000003,C000008,Python数据分析,选修课
```

> 第3行与第1行 (studentId+courseId) 重复，将被去重。

**成功响应** (200):
```json
{
  "code": 200,
  "message": "导入成功，共处理 3 条记录（去重 1 条）",
  "data": {
    "total": 3,
    "duplicated": 1,
    "records": [
      {
        "studentId": "S000001",
        "courseId": "C000001",
        "courseName": "Java程序设计",
        "courseType": "专业课"
      },
      {
        "studentId": "S000002",
        "courseId": "C000003",
        "courseName": "计算机网络",
        "courseType": "公共课"
      },
      {
        "studentId": "S000003",
        "courseId": "C000008",
        "courseName": "Python数据分析",
        "courseType": "选修课"
      }
    ],
    "classified": {
      "专业课": [ ... ],
      "公共课": [ ... ],
      "选修课": [ ... ]
    }
  }
}
```

**错误响应** (400):
```json
{
  "code": 400,
  "message": "CSV格式错误：第3行缺少课程类型字段",
  "data": null
}
```

### 1.3 选课数据检索

按关键词 + 检索类型搜索已导入的选课记录。

| 属性 | 值 |
|------|-----|
| **方法** | `GET` |
| **路径** | `/api/enrollment/search` |

**请求参数** (Query String):

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `keyword` | String | 是 | 检索关键词 |
| `type` | String | 是 | 检索类型：`studentId` / `courseId` / `courseName` / `courseType` |

**type 匹配规则**:
| type 值 | 匹配方式 | 说明 |
|---------|----------|------|
| `studentId` | 精确匹配 | 如 `S000001` |
| `courseId` | 精确匹配 | 如 `C000001` |
| `courseName` | 模糊匹配（包含，忽略大小写） | 如 `Java` 可匹配 "Java程序设计" |
| `courseType` | 精确匹配 | `专业课` / `公共课` / `选修课` |

**请求示例**:
```
GET /api/enrollment/search?keyword=Java&type=courseName
GET /api/enrollment/search?keyword=S000001&type=studentId
GET /api/enrollment/search?keyword=专业课&type=courseType
```

**成功响应** (200):
```json
{
  "code": 200,
  "message": "success",
  "data": [
    {
      "studentId": "S000001",
      "courseId": "C000001",
      "courseName": "Java程序设计",
      "courseType": "专业课"
    }
  ]
}
```

**无匹配结果** (200):
```json
{
  "code": 200,
  "message": "无匹配选课记录",
  "data": []
}
```

## 三、数据模型

### 3.1 选课记录 (EnrollRecord)

```typescript
interface EnrollRecord {
  studentId: string;   // 学生ID，格式：S+6位数字，如 S000001
  courseId: string;    // 课程ID，格式：C+6位数字，如 C000001
  courseName: string;  // 课程名称
  courseType: string;  // 课程类型：专业课 | 公共课 | 选修课
}
```

### 3.2 课程信息 (CourseInfo)

```typescript
interface CourseInfo {
  courseId: string;
  courseName: string;
  courseType: string;
  capacity: number;
  teacherName: string;
  credits: number;
  semester: string;
}
```

### 3.3 分类结果 (ClassifiedResult)

```typescript
interface ClassifiedResult {
  "专业课": EnrollRecord[];
  "公共课": EnrollRecord[];
  "选修课": EnrollRecord[];
}
```

### 3.4 导入结果 (ImportResult)

```typescript
interface ImportResult {
  total: number;                    // 处理后的总记录数
  duplicated: number;               // 去重条数
  records: EnrollRecord[];          // 处理后的记录列表
  classified: ClassifiedResult;     // 按类型分类
}
```

### 3.5 统一响应 (ApiResponse)

```typescript
interface ApiResponse<T> {
  code: number;      // 状态码（200=成功）
  message: string;   // 提示信息
  data: T | null;    // 响应数据
}
```

## 四、前端 TypeScript 调用示例

```typescript
const BASE_URL = 'http://localhost:8080';

// 1. CSV 批量导入
async function importCsv(csvData: string): Promise<ApiResponse<ImportResult>> {
  const formData = new URLSearchParams();
  formData.append('csvData', csvData);
  
  const response = await fetch(`${BASE_URL}/api/enrollment/import`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/x-www-form-urlencoded' },
    body: formData.toString()
  });
  return response.json();
}

// 2. 检索选课数据
async function searchEnrollments(
  keyword: string, 
  type: 'studentId' | 'courseId' | 'courseName' | 'courseType'
): Promise<ApiResponse<EnrollRecord[]>> {
  const params = new URLSearchParams({ keyword, type });
  const response = await fetch(`${BASE_URL}/api/enrollment/search?${params}`);
  return response.json();
}

// 3. 获取样例数据
async function getSampleData(): Promise<ApiResponse<ImportResult>> {
  const response = await fetch(`${BASE_URL}/api/enrollment/sample`);
  return response.json();
}

// 4. 按类型获取
async function getByType(courseType?: string): Promise<ApiResponse<ClassifiedResult>> {
  const url = courseType 
    ? `${BASE_URL}/api/enrollment/by-type?courseType=${courseType}`
    : `${BASE_URL}/api/enrollment/by-type`;
  const response = await fetch(url);
  return response.json();
}

// 5. 导出 Excel
function exportExcel(): void {
  const link = document.createElement('a');
  link.href = `${BASE_URL}/api/enrollment/export`;
  link.download = '';
  document.body.appendChild(link);
  link.click();
  document.body.removeChild(link);
}

// 6. 获取课程列表
async function getCourses(): Promise<ApiResponse<CourseInfo[]>> {
  const response = await fetch(`${BASE_URL}/api/courses`);
  return response.json();
}
```
